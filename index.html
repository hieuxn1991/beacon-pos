🖨 Kết Nối Máy In Nhiệt

Phương thức
✅ Cách đơn giản nhất (không cần driver)
Khi in, app sẽ mở cửa sổ in của trình duyệt.
Chọn máy in nhiệt từ danh sách, chọn khổ giấy đúng (58mm hoặc 80mm), tắt tiêu đề/chân trang, bỏ margin.
1
Cắm máy in vào máy tính/iPad qua cáp USB
2
Chỉ dùng được trên Chrome/Edge trên máy tính (không hỗ trợ iOS Safari)
3
Nhấn "Kết Nối" → chọn máy in trong popup
4
Nếu không thấy máy in: thử chọn nhà sản xuất đúng bên dưới

3
Chỉ dùng được trên Chrome/Edge máy tính Windows/Mac/Linux

💡 Mẹo kết nối máy in
• Đơn giản nhất: Chọn "In qua Browser" — không cần driver, hoạt động trên mọi thiết bị
• Xprinter XP-58/80: Dùng USB mode, chọn Xprinter
• Epson TM-T88: Dùng USB hoặc LAN
• iPad/iPhone: Chỉ có thể dùng Browser print hoặc AirPrint
• Android: Dùng Bluetooth hoặc Browser print

☁️ Đồng Bộ Dữ Liệu

ℹ️ Về tính năng Cloud Sync
Dữ liệu được xuất ra file CSV và bạn upload thủ công lên Google Drive.
Để đồng bộ tự động thật sự, cần deploy app lên server (có hướng dẫn bên dưới).
📥 Xuất & Upload thủ công
Xuất CSV → Upload lên Google Drive folder của bạn
📁 Drive Tre Village 📁 Drive Hải Đăng
📤 Import từ Drive
Tải CSV từ Drive về → Import vào app
🚀 Tự động hoá (Nâng cao)
Để đồng bộ tự động & nhiều thiết bị dùng chung:
1. Host file HTML này lên GitHub Pages (miễn phí)
2. Hoặc dùng Firebase Realtime Database (miễn phí 1GB)
3. Liên hệ developer để tích hợp backend

📱 Cài App lên thiết bị

🍎 iOS / iPad (Safari)
1. Mở file bằng Safari (bắt buộc)
2. Nhấn nút Chia sẻ (□↑)
3. Chọn "Thêm vào Màn hình chính"
4. Nhấn Thêm ✅
🤖 Android (Chrome)
1. Mở bằng Chrome
2. Menu ⋮ → "Thêm vào màn hình chính" ✅
💻 Desktop (Chrome/Edge)
Icon 🖥 ở thanh địa chỉ → Cài đặt ✅
